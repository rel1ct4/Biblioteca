Modelo de dados
1. Usuário
  - Nome -> String
  - email -> String ou symbol
  - CPF -> Number
  - endereço -> String
  - Senha -> String

2. Livros
 - Marca -> String
 - Editora -> String
 - Referência -> number
 - Título -> String
 - Autor -> String
 - Formato -> String
 - Edição -> String
 - Categoria -> String
 - Idioma -> String
 - Número de páginas -> number
 - ISBN -> number
 - Classificação indicativa -> number
 - Quantidade de livros na biblioteca -> number
 - emprestimos totais do livro -> number
 - emprestimos mensais -> number
